INTRODUCTION

Welcome to my HTML project page!


HTML is the standard markup page for the creation of Web pages. HTML stands for Hypertext Markup Language. HTML describes the structure of a Web page and consists of a series of elements. Overall, HTML tells the browser how to display the content on the web page.


ELEMENTS

HTML elements are the contents stored inside the start and end tags. Note: there are some elements that don't require end tags (like "br"), which are also known as empty elements. All HTML pages should be nested HTML elements, which is when elements contain other elements. For example, the "html" tag must contain "head" and "body" starting and ending tags inside the "html" tag.
Some HTML tags don't require an ending tag, but it is always recommended to include them to avoid errors and bugs. HTML is not case sensitive, but it is recommended to use lowercases, at least when it comes to using tags.


ATTRIBUTES

HTML attributes provides additional information to HTML elements. This is available to all HTML elements and are specified in the starting tag. Examples of attributes are "href" (used in hyperlinks), src (used in images), "width" and "height" (used in images), "alt" (used in images"), "style" (used for CSS styling), "lang" (used to declare the default language of the Web page, which can be found in the source code for the "html" starting tag ), "title", etc.
HTML attributes aren't required to be lowercased and surrounded by double quotes, but it is recommended to do so. Single quotes are also allowed in some situations.

This body of text is created as a paragraph using the "p" tag. Paragraphs are used as a block of text in an HTML page. They always start on a new line and there tends to be some margins before and after paragraphs. Sizes of paragraphs can be resized based on the screen size of the user.
The "hr" tag is used to create a horizontal break in an HTML page to seperate content. The "br" tag defines a line break for content to start on a new line. Both tags are empty HTML elements. See them in use in the HTML source code.

CSS stands for Cascading Style Sheets. CSS is used to format the layout of the HTML webpage. It has 3 ways of being added into HTML pages. This is inline CSS, Internal CSS, and External CSS.


Inline CSS applies a CSS style to a single HTML element by using the style attribute on an HTML element as shown below in HTML source code .

Inline CSS


Internal CSS applies a CSS style for the HTML page by defining a style element in the "head" section of the HTML page. See "head" section of HTML source code for example where the background color of the HTML page is changed.


External CSS can define the CSS styles for multiple HTML pags by creating an external file which can be found in the designated CSS file in this GitHub Repository. BLOCK AND INLINE

Every HTML element has a display value. However, in HTML, there are two types of display values. These two types are block and inline.
Block-level elements are elements that alawys start on new lines along with the browser adding a margin before and after the element. Block-level elements typically takes up the full width of the display. The most commonly used block elements are the "p" and "div" tags. Other block elements are the "address", "article", "aside", "blockquote", "canvas", "dd", "div", "dl", "dt", "fieldset", "figcaption", "figure", "footer", "form", "h1-h6", "header", "hr", "li", "main", "nav", "noscript", "ol", "p", "pre", "section", "table", "tfoot", "ul", and "video" tags.
Inline elements are elements that doesn't start on a new line. Unlike block elements, inline elements will only take as much width as it needs. Inline elements in HTML are the "a", "abbr", "acronym", "b", "bdo", "big", "br", "button", "cite", "code", "dfn", "em", "i", "img", "input", "kbd", "label", "map", "object", "output", "q", "samp", "script", "select", "small", "span", "strong", "sub", "sup", "textarea", "time", "tt", and "var" tags.
The "div" element is used in HTML as a container to divide different HTML elements. Attributes aren't required, although, its most common attributes are "id", "class", and "style". Below is an example of the "style" attribute used on a "div" element.


I am Iron Man!


The "span" element is an inline element used to mark up part of a text/document. Similarly to the "div" element, "span" uses the common attributes "id", "class" and "style" attributes although they are not required. Below is an example of the "style" attribute used on a "span" element.


My favorite Spider Man is Tobey Maguire.


CLASSES

The HTML "class" attribute specifies a class for an HMTL element. With the "class" attribute, multiple HTML elements can share the same class. This can also be used in CSS stylesheets and Javascript scripts to access and manipulate elements within the class. To create a class, the syntax starts with a period (.) followed by the class name and curly braces where in between is the properties for the class. The user can have multiple classes for each HTML element and different elements can even share the same class. Note: the class name is case sensitive. In this example, the class will be declared in the head section (found in the HTML source code) and below, the class will be called with the div and span elements below and if done correctly, the example paragraphs will be in the Times New Roman font with a white text color.


Hello World!

Hello Beautiful World!


IDS

The HTML "id" attribute allows the user to specify a unique id for HTML elements. Multiple elements can't have the same id unlike HTML classes. The value of the id attribute must be unique within the HTML page. Just like HTML classes, the "id" attribute can be used in CSS stylesheets and Javascript scripts and the syntax are the same, but instead the "id" attribute uses a hashtag character (#) instead of the period character that the classes use when declaring them. "Ids" are also case sensitive and must have at least one character, can't have spaces, and can't start with a number. In this example, the id will be declared in the head section (HTML source code) and below, the heading will have its font changed to the copperplate font.


Hello World!


Thanks to the "id" attribute, they can serve as bookmarks which can be used to jump from one section of the webpage to another section. This is done by creating an id and then adding a link to the id. When the link is clicked, the page will automatically scroll to the bookmarked link.
